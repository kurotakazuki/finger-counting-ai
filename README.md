# Finger counting AI

手の指の形（ハンドサイン）によって、手軽に演奏できるアプリ。

## 演奏方法

| 音程・コマンド | ハンドサイン            |
| -------------- | ----------------------- |
| ド             | 1 人差し指              |
| ...            | ...                     |
| ソ             | 5 全部の指を Open       |
| ラ             | 6 親指のみ Open         |
| シ             | 7 親指と人差し指を Open |
| 偶数のアタック | 小指のみを Open         |
| ミュート       | グー                    |
